# Deployment Targets

All targets use the same app behavior:
- app binds to `0.0.0.0`
- app listens on `PORT` (default `8080`)

This folder contains provider-specific templates and notes only.
It is not used by the app during local development.

Use one of these provider guides:
- Azure: `deploy/azure/README.md`
- GCP: `deploy/gcp/README.md`
- AWS: `deploy/aws/README.md`
